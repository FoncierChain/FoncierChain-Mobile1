# ENVIRONNEMENT PRO - FONCIERCHAIN 2026

## Configuration
- **Base URL**: `https://api-pro.foncierchain.cg/api/v1`
- **Blockchain**: Mainnet Gov-Congo
- **Security**: HSM (Hardware Security Module) pour les signatures.
- **Compliance**: Loi foncière 2026 (Tripartite mandatory).

## Accès
L'accès nécessite un token physique et une vérification KYC biométrique effectuée au préalable.
